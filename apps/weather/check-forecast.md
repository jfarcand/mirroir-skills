Open Weather and extract current conditions and the 10-day forecast. The AI reads temperature, conditions, and daily highs/lows from the screen — no Weather API key needed.

## Steps

1. Launch **Weather**
2. Wait for "Weather" to appear
3. Remember: Read the current temperature, conditions (sunny/cloudy/rainy), and location name.
4. Screenshot: "current_weather"
5. swipe: "up"
6. Wait for "10-Day Forecast" to appear
7. Remember: Read the 10-day forecast — for each day, note the high, low, and conditions.
8. Screenshot: "ten_day_forecast"
