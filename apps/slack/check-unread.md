Open Slack, check the Activity feed, and summarize unread notifications. The AI reads message previews and channel names from the screen — no Slack API token needed.

## Steps

1. Launch **Slack**
2. Wait for "Home" to appear
3. Tap "Activity"
4. Wait for "Activity" to appear
5. Remember: Read all visible unread notifications — include channel names, sender names, and message previews.
6. Screenshot: "unread_activity"
