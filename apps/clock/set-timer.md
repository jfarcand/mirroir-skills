Start a countdown timer in the Clock app. Navigates to the Timer tab, starts the default timer, and verifies it's running by checking for the Pause button.

## Steps

1. Launch **Clock**
2. Wait for "Timer" to appear
3. Tap "Timer"
4. Wait for "Start" to appear
5. Tap "Start"
6. Wait for "Pause" to appear
7. Verify "Pause" is visible
8. Screenshot: "timer_running"
